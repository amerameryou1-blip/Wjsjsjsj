# Simple Linux Controller Bundle

This repository stores a simple Kaggle-ready bundle and launcher.

## Easy steps

1. Open the website.
2. Paste your GitHub token in the upload box.
3. Click **Upload bundle to GitHub**.
4. Click **Copy kaggle_launcher.py**.
5. Paste the launcher into one Kaggle notebook cell.
6. Run the cell.
