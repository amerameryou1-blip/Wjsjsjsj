# Kaggle Bundle Uploader

This repository stores a Kaggle-ready Python bundle and launcher.

## What you do

1. Open the website.
2. Paste a GitHub token with repo write access.
3. Click **Upload bundle to GitHub**.
4. Copy `kaggle_launcher.py` from the website.
5. Paste that launcher into a Kaggle notebook cell.
6. Run the cell.

That is the intended flow.

## Launcher notes

`kaggle_launcher.py` is designed to run in Kaggle.
It installs the small Python dependencies it needs, fetches the full bundle from GitHub, caches the files, and starts the dashboard.

## GitHub upload notes

Paste your token into the website upload section and click the upload button.
